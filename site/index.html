The daemon that gives
Claude a life

Multi-agent messaging daemon built on the Claude SDK.
Connect Claude to WhatsApp, Telegram, and Discord — with session routing, automation, and embedded NATS JetStream.

Everything you need

From message routing to outreach campaigns — one daemon, zero external services.

channels

Multi-Channel

WhatsApp, Telegram, Discord via omni. Connect multiple accounts to different agents.

agents

Multi-Agent

Multiple agents with different models, modes, and permissions. Active or sentinel mode.

sessions

Session Routing

Per-peer, per-group, per-thread isolation. Cross-session messaging between agents.

cron

Cron Jobs

Schedule prompts with cron expressions, intervals, or one-shot times. Timezone-aware.

triggers

Event Triggers

Subscribe to NATS topics and fire agent prompts. Template variables, filters, cooldowns.

outbound

Outbound Queues

Automated outreach campaigns with follow-ups, qualification tracking, and humanized delivery.

heartbeat

Heartbeat

Proactive agent runs on schedule. Reads HEARTBEAT.md and acts on pending tasks.

permissions

REBAC Permissions

Relation-based access control for tools, executables, sessions, and contacts.

media

Media Pipeline

Audio transcription (Whisper), video analysis (Gemini), image generation, media sending.

Up and running in 60 seconds

1

Install

git clone https://github.com/filipexyz/ravi.git && cd ravi
bun install && bun run build && bun link
2

Setup

ravi setup

Downloads nats-server, configures Claude auth, creates your first agent.

3

Start the daemon

ravi daemon start
4

Connect a channel

ravi whatsapp connect

Scan the QR code and start chatting with Claude on WhatsApp.
